# Fruit Festia

A full-stack, e-commerce web application designed for browsing, selecting, and ordering fresh fruits. This project features user authentication, a structured relational database layout, and dynamic data handling.

---

## Features

- **User Authentication:** Secure user registration, sign-up, and session-based login validation.
- **Dynamic Data Operations:** Complete interaction with a backend relational database to handle customer accounts and transaction billing processing.
- **Modular Architecture:** Clean division of responsibilities with a dedicated source tree grouping complex PHP routing routines and imagery.
- **Responsive User Interface:** Lightweight, cross-browser compatible markup ensuring accessibility across mobile devices and desktop computers.

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP (Session tracking, request handling)
- **Database:** MySQL (Relational tables, structure dumps)
- **Hosting/Environment:** XAMPP (Local development), InfinityFree (Production hosting via Apache)
